# Monitoramento de Nível de Água
Repositório feito para guardar o código do nosso projeto de monitorador de nível de água

## Materiais
ESP32 \
LCD 16 x 2 \
Sensor ultrasônico HCSR04 \
caixinha de mdf \
pote retangular

## Calculo de Volume
Sabendo a altura, comprimento e largura do recipiente, basta calcular o volume da área vazia e subtrair da área total, assim conseguimos o volume da água (se o recipiente for irregular, é preciso realizar outros cálculos de volume)
